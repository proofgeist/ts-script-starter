# ts-script-starter

Node 22.7 Typescript starter

## Create new project

use degit

```bash
npx degit proofgeist/ts-script-starter my-project
```

## copy env-starter to .env

```bash
cp env-starter .env
```

## Install dependencies

```bash
npm install
```

## Run

```bash
npm run ts
```

## Tests

```bash
npm run test
```

Test setup is from
https://nodejs.org/en/learn/test-runner/using-test-runner
